Routine Simulation

In this section, you have to drag a descriptive file of routines of a user, and by modifying the threshold parameter (which determines the variability of the time slots of the routines), you can obtain a file in which for each minute of the day you have noted in which room a user is in.

This file can be obtained in the Routines Definition section.

Select or drag a file to process it and continue working with the web page.
