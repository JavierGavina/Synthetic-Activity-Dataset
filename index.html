User Guide

This section provides a comprehensive guide for users to get acquainted with the platform.

Room Assignment

In this section you can interactively define the rooms of a house and obtain a descriptive file of rooms in JSON format.

Routine Definition

In this section you can interactively define the routines followed by a user and obtain a descriptive file of routines in JSON format.

Routine Simulation

In this section you can put a descriptive file of routines in JSON format, which you can define previously in the previous section, and by means of a simulation algorithm, obtain a file in which it indicates in a year in which rooms the user is in each minute of the day.
