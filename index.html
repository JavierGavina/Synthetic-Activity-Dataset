Weekly Routines

Explanation

Here you have to indicate from what time to what time the user is in one room or another.

Meaning of each field:

  • Weekly Day: Means the day of the week
  • Proportion: The proportion of times that the user does the activity per month:
    • 25: One time per month
    • 50: Two times per month
    • 75: Three times per month
    • 100: All weekly days per month
  • Room: Room where the user is
  • Start Time: Time when the user enters the room
  • End Time: Time when the user leaves the room
